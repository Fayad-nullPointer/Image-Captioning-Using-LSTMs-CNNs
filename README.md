# Image Captioning Project: CNNs, LSTMs, and Advanced Dual Learning Approaches

This repository contains a comprehensive deep learning project for generating natural language descriptions of images (Image Captioning). The project is divided into two main components, showcasing a standard state-of-sart approach alongside an experimental "Dual-Task" learning approach. 

The primary dataset used is the **Flickr8k Dataset**.

## 📁 Repository Structure

* **`notebook_FINAL.ipynb`**: The primary, stable notebook containing the end-to-end pipeline for the standard Attention-based Encoder-Decoder model.
* **`New_Approaches.ipynb`**: An experimental notebook extending the standard model with a **Dual-Task Learning** approach (simultaneous Denoising and Captioning).
* **`evaluate_metrics.py`**: A specialized evaluation script for calculating various NLP model metrics.


## 🧠 Core Architecture (`notebook_FINAL.ipynb`)

The foundational architecture is based on the widely adopted **Encoder-Decoder with Attention** paradigm.

### 1. Encoder (CNN)
* Utilizes a pre-trained **ResNet-50** model.
* Acts as a feature extractor to capture rich spatial and semantic representations from the input images.
* The fully connected classification layers are removed, and a custom Convolutional layer maps the ResNet features into the desired embedding dimension.

### 2. Attention Mechanism
* Implements a **Soft Attention** (Bahdanau-style) mechanism.
* Allows the decoder to selectively focus on different semantic regions of the image while generating each word of the caption, mimicking human visual attention.

### 3. Decoder (RNN)
* Built upon an **LSTM (Long Short-Term Memory)** network.
* Conditioned on the context vector generated by the Attention mechanism and the previous word embedding to predict the next word in the sequence.


## 🔥 Advanced Approaches (`New_Approaches.ipynb`)

This notebook pushes the boundaries by introducing a **Dual Learning Objective** to regularize the model and force it to learn more robust visual representations.

### The Dual-Task Denoising Architecture
* **Synthetic Noise:** Random Gaussian noise is synthetically added to the input images during training.
* **Shared Encoder:** The noisy images are passed through the ResNet-50 encoder.
* **Task 1: Denoising Decoder:** A secondary transpose-convolutional decoder network takes the encoded features and attempts to reconstruct the original, clean image (MSE Loss).
* **Task 2: Captioning Decoder:** Simultaneously, the attention-based LSTM decoder generates the image caption (Cross-Entropy Loss).

**Why Dual Learning?**
Forcing the encoder to maintain sufficient fine-grained information to reconstruct the image prevents it from dropping crucial spatial details. This regularizes the network, improves general-purpose visual understanding, and ultimately aims to yield better, grounded captions.

## 📊 Evaluation

Both approaches are evaluated using standard Natural Language Generation metrics:
* **BLEU Score** (Bilingual Evaluation Understudy): Assesses the n-gram overlap between the generated captions and the ground-truth references. Evaluated across BLEU-1, BLEU-2, BLEU-3, and BLEU-4.
